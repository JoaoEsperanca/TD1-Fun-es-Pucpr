Documentação – TDE-01: Funções em JavaScript Objetivo Este projeto tem como objetivo demonstrar o uso de funções em JavaScript, explorando: Criação de funções nomeadas; Uso de funções anônimas; Utilização de arrow functions; Aplicação de funções como parâmetro (callbacks). A atividade foi desenvolvida como parte da disciplina Resolução de Problemas Estruturados em Computação.
1. Funções Matemáticas Básicas Foram criadas quatro funções nomeadas para operações aritméticas simples: soma(a, b) → retorna a soma dos dois números. subtracao(a, b) → retorna a diferença entre os números. multiplicacao(a, b) → retorna o produto entre eles. divisao(a, b) → retorna o quociente, com verificação para evitar divisão por zero. Essas funções mostram como declarar e reutilizar código de forma modular e organizada.
2. Manipulação de Arrays Um array chamado numeros foi criado com valores de 1 a 10. Foram aplicadas duas funções para demonstrar manipulação de dados: filter() com função anônima para selecionar apenas os números pares; reduce() com arrow function para calcular a média dos valores. Esses exemplos mostram como funções podem ser passadas como parâmetros de métodos internos do JavaScript.
3. Função Callback A função calcular(a, b, operacao) foi criada para receber outra função como argumento (callback). Exemplo de uso: calcular(5, 7, soma) → executa a soma; calcular(5, 7, multiplicacao) → executa a multiplicação. Isso demonstra o conceito de função de ordem superior, muito usado em JavaScript moderno.
4. Arrow Functions Além de reduce(), outra arrow function foi usada no método map(): const quadrados = numeros.map(num => num * num) Gera um novo array com o quadrado de cada número. Arrow functions tornam o código mais conciso e legível.
Conclusão
O projeto cumpre todos os requisitos da atividade: Usa funções nomeadas, anônimas e arrow functions; Implementa uma função callback; Manipula arrays; Apresenta código organizado, funcional e testável no navegador. O resultado final pode ser visualizado pelo console do navegador, ao abrir o arquivo index.html.
